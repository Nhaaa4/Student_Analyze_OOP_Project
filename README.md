# Student Performance and Learning Style

## Introduction
    This proposal addresses the final project for the Data Science department at the Cambodia Academy of Digital Technology. It focuses on enhancing education support systems through the analysis of personalized student performance data and learning styles. By examining these factors, we aim to better understand how to support students effectively and improve their educational experiences.
## Installation
    This project is written in **Python**, so ensure you have the **latest version** of Python installed. If you encounter issues with certain libraries, try switching to Python **13.12.7** for compatibility.
### Step 1: Check Python Version
    Verify your Python installation by running:  
    ```bash
    python --version
    ```
    If needed, download from Python from python.org
### Step 2: Install Required Package
    You need to use the package manager [pip](https://pip.pypa.io/en/stable/) to install pandas, seaborn, matplotlib.
    ```bash
    pip install pandas seaborn matplotlib
    ```
